Online Retail Dataset - Machine Learning Lab

About : Data analysis project on Online Retail Dataset using Python and Pandas. 

Dataset 
File: Online Retail.xlsx 
Rows: 5,36,641 
Columns: 10

Tasks Performed 
1. Load data and display 10 rows
2. Check rows and missing values
3. Remove duplicate values
4. Add Revenue column
5. Top 10 selling products by quantity
6. Calculate total revenue of the store
7. Total number of unique customers
8. Calculate average order value
9. Country with highest number of orders
10. Products bought together
11. Customers who purchased more than 10 items
12. Peak shopping hours
